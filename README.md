# JSON Server

it can be used to create a mock REST API for prototyping websites.

1.  clone the repository:
    ```shell script
    git clone https://github.com/Bardiesel/json-server.git
    ```
2.  install packages:
    ```shell script
    npm install -g json-server
    ```
3.  start json server:
    ```shell script
    json-server --watch db.json
    ```
